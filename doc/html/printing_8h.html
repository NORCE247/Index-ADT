Detailed Description

Defines printing macros.

Author
Morten Grønnesby (morte.nosp@m.n.gr.nosp@m.onnes.nosp@m.by@u.nosp@m.it.no)
Version
0.1

The file defines a number of printing macros that prints with color to different streams. The different macros can be turned on and off using log level at compile time.

The printing functions prints to different streams depending on their purpose. This means that it is possible to redirect different streams to files and still have debug or info messages.

Example:

./index data/ 2> debug.txt
