Destroys the given map. Will also destroy keys and values if functions are supplied.

Parameters
mapThe map to destroy.
destroy_keyA pointer to a function to destroy the keys. Can be NULL.
destroy_valA pointer to a function to destroy the values. Can be NULL.

The function will destroy keys if the supplied destroy_key function is supplied. The function will destroy values if the supplied destroy_values function is supplied. If the arguments are NULL, the respective field is not touched.
